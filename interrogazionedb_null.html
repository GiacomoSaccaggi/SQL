Missing value

In SQL, NULL rappresenta un valore mancante o sconosciuto, è possibile verificare la presenza di valori NULL utilizzando l'espressione IS NULL. Ad esempio, per contare il numero di valori mancanti in una colonna:

SELECT COUNT(*)
FROM tabella
WHERE nome_variabile IS NULL;

A volte, si vorrà filtrare i valori mancanti in modo da ottenere solo risultati che non sono NULL. Per fare questo, potete usare l'operatore IS NOT NULL:

SELECT *
FROM tabella
WHERE nome_variabile IS NOT NULL;
